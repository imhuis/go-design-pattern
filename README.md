# go-design-pattern

## 分类

1. 创建型模式
    - 单例模式（Singleton）
    - 原型模式（Prototype）
    - 工厂方法模式（Factory Method）
    - 抽象工厂模式（Abstract Factory）
    - 建造者模式/生成器（Builder）
2. 结构型模式
    - 适配器模式（Adapter）
    - 桥接模式（Bridge）
    - 组合模式（Composite）
    - 装饰模式（Decorator）
    - 外观模式（Facade）
    - 享元模式（Flyweight）
    - 代理模式（Proxy）
3. 行为模式
    - 责任链模式（Chain of Responsibility）
    - 命令模式（Command）
    - 迭代器模式（Iterator）
    - 中介者模式（Mediator）
    - 备忘录模式（Memento）
    - 观察者模式（Observer）
    - 状态模式（State）
    - 策略模式（Strategy）
    - 模板方法模式（Template Method）
    - 访问者模式 （Visitor）
